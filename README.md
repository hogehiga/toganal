#Toganal
##What's it
Print the word which has been contained in tweets at togetter page, with rank order of frequency of use.

##Usage
    ruby toganal.rb some_togetter_page.html

##Example
    $ ruby toganal.rb 270512.html
    "Java: 375"
    "が: 357"
    "屋: 341"
    "単語: 337"
    "懐かしく: 336"
    "感じる: 335"
